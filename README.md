This program is a simple project tracker that asks for your project title and group name, shows a list of predefined tasks with their estimated hours, and then lets you assign up to four tasks to group members while marking each as “done” or “pending.” Each task is given points (2 if done, 1 if pending), and all assignments are stored in a list; after that, the program calculates the total points and converts it into a percentage to measure progress, then labels the project status as “PROJECT READY,” “ON TRACK,” or “NEEDS MORE WORK” based on that percentage, and finally displays everything in a neat table showing the tasks, assigned members, their status, points, and the overall progress of the project.
